# simple-logger

编写代码在目录test/demo/下 </br>
log4sh已经编译过了 </br>
运行是目录在/root </br>
##程序清单
***
log4sh log 框架 </br>
demo.sh 主程序 </br>
file.log 主系统日志文件 </br>
log_halt.sh 关机前需要执行的脚本 </br>
logout.log 文件清空日志记录 </br>
who.old 存储先前的在线用户信息 </br>
who.new 存储当前在线用户信息 </br>
service.old 存储先前的服务状态 </br>
service.new 存储当前的服务状态 </br>
use_ratio.old 存储先前的资源占用率 </br>
